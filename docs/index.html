Projects

Computational Intelligence Framework for Python

Homepage

CIFY is a framework for common computational intelligence paradigms written in Python. The framework provides end users with the ability to easily and reliably implement nature-inspired metaheuristics. A collection of well-known metaheuristics are provided alongside tools to manipulate these metaheuristics, define objective functions for them to optimize, and create pipelines out of collections of varying approaches.
